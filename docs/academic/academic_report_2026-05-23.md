- **发表**: 2026-05-15T17:49:24+00:00
- **最终评分**: 9.0/10

- **核心发现**: 该研究通过实证和理论分析，揭示了生成式AI在人类在线交流中作为中介时，如何通过编辑文本引入方向性偏见，并经由社交网络放大，最终引导集体意见向特定方向偏移。研究发现，多个流行LLM家族在编辑关于有争议话题（如枪支管控、无神论）的文本时，表现出系统性偏向（如支持枪支管控、反对无神论）。作者构建了意见动态数学模型，在真实社交网络数据上模拟，证实AI偏见可通过网络放大并改变集体意见。此外，对X平台'解释此帖'功能的审计发现，Grok在堕胎相关内容上存在'亲生命'偏见，并可追溯到具体设计选择。这些发现表明，AI中介通信可能成为操纵集体意见的潜在工具，对在线平台的民主讨论构成风险。研究讨论了其对欧盟监管立法的影响。
