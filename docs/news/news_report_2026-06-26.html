对 GPT-OSS、Qwen、Olmo 和 Phi 系列前沿开源推理模型的研究发现，所谓的「思考 token」并未带来真正的安全性深思熟虑。模型是否拒绝或服从指令在第一个 token 阶段就已决定，后续的「思考链」主要扮演事后合理化角色而非真正的安全推理过程。系统性检验了推理模型的「表面对齐」现象——模型看起来在「思考」安全决策，但实际上安全判断在推理过程之前就已做出。对当前以 CoT 为核心的安全对齐策略提出根本性质疑。
强烈支持「进化对齐脆弱性」——思考 token 的安全防护是表面化的，对齐在推理阶段即可被绕过。直接反驳「推理模型=更安全模型」的主流叙事，支撑「共识牢笼」——行业共识（CoT 带来安全）被实证击穿。还关联「认知金融化/Token陷阱」——思考 token 被包装为安全机制出售，但实际只是表面功夫。该论文是进化对齐脆弱性模型的关键实验证据。
